# Lockdown Sound Effect

This directory contains sound effects for the Stark Industries Operating System.

## Files:
- lockdown.mp3 - Lockdown alarm sound effect

## Usage:
The lockdown sound is automatically played when the LOCKDOWN defense mode is activated in the Security module.

## Technical Details:
- Format: MP3
- Duration: ~1 second
- Volume: 70% 
- Priority: High (security alert)

## Note:
If the audio file is missing, the system will fallback to Web Audio API generated sounds.
